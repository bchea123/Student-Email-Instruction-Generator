Step 1:
Install latest version of Python using the link below.

	Latest version of Python as of 4/7/2021 (3.9.4)
	https://www.python.org/downloads/release/python-394/

Step 2:
Install Python modules using the Command Line with the following commands for each module.

	gspread
		pip install gspread

	docx
		pip install python-docx

	docx2pdf
		pip install docx2pdf

	oauth2client
		pip install oauth2client

Step 3:
Run python application and follow requests for user input.
